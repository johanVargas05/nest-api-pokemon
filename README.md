# Ejecutar en desarrollo
1. Clonar el repositorio
2. ejecutar
```
yarn install
```
3. Tener NEST JS CLI instalado en dado caso de no tenerlo ejecutar el siguiente comando: 
```
npm i -g @nestjs/cli
``` 
4. Levantar la base de datos 
```
docker-compose up -d
```

## Stack usado
* MongoDB
* Nest js
